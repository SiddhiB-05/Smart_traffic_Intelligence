# 🚦 Smart Traffic Intelligence System

An AI-powered traffic forecasting and decision-support platform that helps authorities predict congestion caused by planned events and detect unexpected traffic anomalies before they become major disruptions.

## ✨ Features

* Event-based traffic congestion prediction
* Congestion heatmap visualization
* Diversion route recommendations
* Traffic police and barricade planning
* Anomaly detection for unexpected traffic surges
* Interactive dashboard for monitoring and alerts

## 🧠 Machine Learning

### Traffic Prediction

* XGBoost / LightGBM
* Predicts traffic volume, congestion level, and risk score

### Anomaly Detection

* Isolation Forest / Statistical Thresholding
* Detects unexpected congestion and crowd surges
